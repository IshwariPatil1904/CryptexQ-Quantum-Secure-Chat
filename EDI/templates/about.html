About CryptexQ

CryptexQ is an academic project that simulates Quantum Key Distribution (QKD) with AES-GCM encryption. It demonstrates how secure communication can be achieved using quantum principles without real quantum hardware.

Scope

  • Secure text messaging using simulated QKD.
  • Interactive educational experiments for students and enthusiasts.
  • Visualization of qubit generation, basis selection, and measurement.
  • Support for workshops, seminars, and classroom learning.
  • Foundation for future extension to PQC or real QKD hardware integration.

Objectives

  • Simulate the BB84 QKD protocol step-by-step.
  • Generate symmetric keys for secure encryption.
  • Encrypt and decrypt messages using AES-GCM.
  • Provide an interactive and educational interface.
  • Bridge theory with hands-on experimentation.
